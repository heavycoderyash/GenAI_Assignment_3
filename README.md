# AI Agent CLI - Scaler Academy Frontend Builder

A conversational Command Line Interface (CLI) tool powered by an autonomous AI agent. This tool takes natural language instructions and builds a high-fidelity frontend clone of the Scaler Academy homepage using HTML, CSS and JavaScript. 

Built for GenAI Assignment 2.

## 🚀 Features

* **Autonomous ReAct Loop:** The agent uses a continuous `THINK` -> `ACTION` -> `OBSERVE` reasoning loop to break down complex UI tasks.
* **Multi-File Generation:** Automatically generates perfectly linked `index.html`, `styles.css`, and `script.js` files.
* **Smart Parsing:** Uses a bulletproof text-extraction system to safely parse AI-generated code without JSON breaking or Markdown interference.
* **Auto-Execution:** Detects your operating system and automatically opens the generated website in your default browser once the build is complete.
* **Streaming Output:** Features a sleek typewriter effect in the terminal so you can watch the agent's thought process in real-time.

## 🛠️ Tech Stack

* **Runtime:** [Node.js](https://nodejs.org/) (ES Modules)
* **AI Model:** Any model is fine
* **SDK:** OpenAI Node.js SDK
* **Other Packages:** `dotenv`

## ⚙️ Prerequisites

Before you begin, ensure you have met the following requirements:
* Node.js (v18 or higher) installed on your machine.
* An API key of any Language model.

## 💻 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/heavycoderyash/GenAI_Assignment_2.git
   cd GenAI_Assignment_2
   ```

2. **Install dependencies:**
   ```bash
   npm install openai dotenv
   ```

3. **Environment Variables:**
   Create a `.env` file in the root directory and add your API key:
   ```env
   API_KEY = api_key_here
   BASE_URL = base_url_here
   MODEL = model_here
   ```

4. **Package.json Configuration:**
   Ensure your `package.json` includes `"type": "module"` to support ES6 imports.

## 🏃‍♂️ How to Run

1. Start the CLI Agent:
   ```bash
   node index.js
   ```

2. When the prompt `>>` appears, type your instruction. For example:
   > *Clone Scaler Academy website.*

3. Sit back and watch the terminal. The agent will output its thought process, create the necessary files in your directory, and automatically launch the final webpage in your browser.
